# Green for Life Rwanda - Website Redesign

## Project Name
Green for Life Rwanda Official Website

## Purpose
This is a complete redesign of the existing website for **Green for Life Rwanda**, an environmental NGO dedicated to conservation and sustainable development in Rwanda. The new site is modern, clean, responsive, and built with static HTML, CSS, and JavaScript.

## Pages
- Home (index.html)
- About (about.html)
- Programs (programs.html)
- Projects (projects.html)
- Impact (impact.html)
- News (news.html)
- Get Involved (get-involved.html)
- Contact (contact.html)
